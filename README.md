 # Institut Le Xaragua

Site web vitrine de l’Institut Le Xaragua réalisé dans le cadre du TD 2 :  
**Création collaborative d’un site web avec Git & GitHub**

---

## Description

Ce projet consiste à développer un site web statique présentant un institut de formation universitaire et professionnelle.

Le site a été réalisé en utilisant une approche collaborative avec Git et GitHub, incluant la gestion des branches, les Pull Requests et la résolution de conflits.

---

##  Structure du site

Le site est composé de trois pages principales :

- **Accueil (index.html)**  
  Présentation de l’institut et de sa mission.

- **Formations (formations.html)**  
  Liste des programmes universitaires et professionnels.

- **Contact (contact.html)**  
  Formulaire de contact et informations de l’institution.

##  Méthodologie de travail

Le projet a été réalisé selon les étapes suivantes :

1. Initialisation du dépôt Git
2. Création de branches :
   - `feature/formations`
   - `feature/contact`
3. Développement parallèle des fonctionnalités
4. Utilisation des Pull Requests pour fusionner le code
5. Simulation d’un conflit sur le fichier `style.css`
6. Résolution du conflit et validation finale

---

##  Gestion des conflits

Un conflit a été volontairement créé lors de la modification du fichier CSS par deux développeurs.

Ce conflit a été résolu en :
- identifiant les différences
- fusionnant les modifications
- supprimant les marqueurs de conflit
